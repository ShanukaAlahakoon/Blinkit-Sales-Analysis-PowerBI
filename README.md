# 🛒 Blinkit Sales Analysis (Power BI)

## 📌 Project Overview
This project focuses on analyzing Blinkit's sales performance, customer satisfaction, and inventory distribution using Power BI.

## 🎯 Business Objective
To identify key insights and opportunities for optimization using KPIs and interactive dashboards.

## 📊 Key KPIs
- Total Sales
- Average Sales
- Number of Items
- Average Rating

## 📈 Dashboard Insights
- Tier 3 outlets generate higher sales compared to Tier 1 and Tier 2
- Fruits and Snack items are top-performing categories
- Outlet size has a direct impact on sales distribution
- Sales peaked around 2018 based on outlet establishment trends

## 📂 Dataset
- BlinkIT Grocery Data.csv

## 🛠 Tools Used
- Power BI
- DAX
- Data Visualization
